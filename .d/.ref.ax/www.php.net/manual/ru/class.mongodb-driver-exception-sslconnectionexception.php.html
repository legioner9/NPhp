    Класс MongoDB\Driver\Exception\SSLConnectionException (устаревший)

    (mongodb >= 1.0.0)

    Внимание

    Этот класс исключений устарел и может быть удалён в будущей основной версии. Драйвер никогда не выбросит это исключение.

    Введение

    Выбрасывается, когда драйверу не удаётся установить SSL-соединение с сервером.
