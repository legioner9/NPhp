    DOMAttr::__construct

    (PHP 5, PHP 7, PHP 8)

    DOMAttr::__construct — Создаёт экземпляр класса DOMAttr

    Описание

    public DOMAttr::__construct(string $name, string $value = "")

    Создаёт новый объект класса DOMAttr. Этот объект будет доступен только для чтения. Он может быть добавлен к документу, но дополнительные узлы нельзя добавить к этому узлу, пока он присоединён к документу. Для создания узла с возможностью модификации используйте DOMDocument::createAttribute.

    Список параметров

    name

    Имя тега атрибута.

    value

    Значение атрибута.

    Примеры

    Пример #1 Создание нового объекта класса DOMAttr

    <?php

    $dom     = new DOMDocument('1.0', 'iso-8859-1');
    $element = $dom->appendChild(new DOMElement('root'));
    $attr = $element->setAttributeNode(new DOMAttr('attr', 'attrvalue'));
    echo     $dom->saveXML();

    ?>

    Результат выполнения приведённого примера: 

    <?xml version="1.0" encoding="utf-8"?>
<root attr="attrvalue" />
