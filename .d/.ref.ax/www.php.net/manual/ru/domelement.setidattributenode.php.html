    DOMElement::setIdAttributeNode

    (PHP 5, PHP 7, PHP 8)

    DOMElement::setIdAttributeNode — Объявляет указанному узлу атрибута тип ID

    Описание

    public DOMElement::setIdAttributeNode(DOMAttr $attr, bool $isId): void

    Объявляет атрибут, определённый узлом attr, уникальным идентификатором элемента.

    Список параметров

    attr

    Узел атрибута.

    isId

    Параметр устанавливают в true, если нужно, чтобы тип переданного атрибута с именем name стал идентификатором элемента, иначе указывают false.

    Возвращаемые значения

    Функция не возвращает значения после выполнения.

    Ошибки

    DOM_NO_MODIFICATION_ALLOWED_ERR

    Возникает, если узел доступен только для чтения.

    DOM_NOT_FOUND

    Возникает, если атрибут с именем name не принадлежит элементу.

    Смотрите также

    • DOMDocument::getElementById() - Ищет элемент с определённым идентификатором
    • DOMElement::setIdAttribute() - Объявляет атрибуту c указанным именем тип ID
    • DOMElement::setIdAttributeNS() - Объявляет атрибуту с указанными локальным именем и URI пространства имён тип ID
