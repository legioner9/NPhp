    Ds\Map::get

    (PECL ds >= 1.0.0)

    Ds\Map::get — Возвращает значение по ключу

    Описание

    public Ds\Map::get(mixed $key, mixed $default = ?): mixed

    Возвращает значение по заданному ключу, или необязательное значение по умолчанию, если ключ не найден.

    Замечание:

    Поддерживаются значения типа object. Если объект реализует интерфейс Ds\Hashable, проверка производится путём вызова метода объекта equals. Если объект не реализует интерфейс Ds\Hashable, объекты должны ссылаться на один и тот же экземпляр класса.

    Замечание:

    Вы можете использовать синтаксис массива для доступа к значениям, т.е. $map["key"].

    Предостережение

    Будьте осторожны при использовании синтаксиса массива. Скалярные ключи будут приведены к целым движком PHP. К примеру, $map["1"] будет пытаться обратиться к int(1), тогда как $map->get("1") обратится к правильному элементу.

    Смотрите раздел Массивы.

    Список параметров

    key

    Ключ.

    default

    Необязательное значение, которое вернётся, если элемент с заданным ключом отсутствует.

    Возвращаемые значения

    Значение по ключу key, или значение default, если ключ не найден.

    Ошибки

    Выбрасывает исключение OutOfBoundsException если ключе не найден и значение по умолчанию не задано.

    Примеры

    Пример #1 Пример использования Ds\Map::get()

    <?php
    $map     = new \Ds\Map(["a" => 1, "b" => 2, "c" => 3]);

    var_dump($map->get("a")); // 1
    var_dump($map->get("d", 10)); // 10 (используется по умолчанию)
    ?>

    Вывод приведённого примера будет похож на: 

    int(1)
int(10)

    Пример #2 Пример использования Ds\Map::get() с синтаксисом массива

    <?php
    $map     = new \Ds\Map(["a" => 1, "b" => 2, "c" => 3]);

    var_dump($map["a"]); // 1
    ?>

    Вывод приведённого примера будет похож на: 

    int(1)
