    Event::timer

    (PECL event >= 1.2.6-beta)

    Event::timer — Создать объект события таймера

    Описание

    public static Event::timer( EventBase $base , callable $cb , mixed $arg = ?): Event

    Создаёт объект события таймера. Это облегчённый метод для создания события сигнала. Обратите внимание, что штатный конструктор Event::__construct() также может создавать события таймера.

    Список параметров

    base

    Событийная база, к которой необходимо привязать событие.

    cb

    Функция обратного вызова события таймера. Смотрите Функции обратного вызова для событий .

    arg

    Пользовательские данные, связанные с событием. Они будут переданы в функцию обратного вызова когда произойдёт событие.

    Возвращаемые значения

    Возвращает объект Event. В случае возникновения ошибки возвращает false.
