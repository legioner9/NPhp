    Предопределённые константы

    Модуль определяет перечисленные ниже константы и открывает к ним доступ только тогда, когда модуль либо собрали в PHP, либо динамически загрузили во время исполнения кода.

    INPUT_POST (int)
    Переменные POST.
    INPUT_GET (int)
    Переменные GET.
    INPUT_COOKIE (int)
    Переменные COOKIE.
    INPUT_ENV (int)
    Переменные ENV.
    INPUT_SERVER (int)
    Переменные SERVER.
    INPUT_SESSION (int)
    Переменные SESSION. (ещё не реализовано)
    INPUT_REQUEST (int)
    Переменные REQUEST. (ещё не реализовано)
    FILTER_FLAG_NONE (int)
    Не использовать флаги.
    FILTER_REQUIRE_SCALAR (int)
    Флаг используется для указания необходимости скалярного значения в качестве входного значения.
    FILTER_REQUIRE_ARRAY (int)
    Требовать массив в качестве входного значения.
    FILTER_FORCE_ARRAY (int)
    Всегда возвращать массив.
    FILTER_NULL_ON_FAILURE (int)
    Использовать NULL вместо FALSE в случае возникновения ошибки.
    FILTER_VALIDATE_INT (int)
    Идентификатор фильтра "int".
    FILTER_VALIDATE_BOOL (int)
    Псевдоним FILTER_VALIDATE_BOOLEAN.
    FILTER_VALIDATE_BOOLEAN (int)
    Идентификатор фильтра "boolean".
    FILTER_VALIDATE_FLOAT (int)
    Идентификатор фильтра "float".
    FILTER_VALIDATE_REGEXP (int)
    Идентификатор фильтра "validate_regexp".
    FILTER_VALIDATE_URL (int)
    Идентификатор фильтра "validate_url".
    FILTER_VALIDATE_DOMAIN (int)
    Идентификатор фильтра "validate_url". (Доступно с PHP 7.0.0)
    FILTER_VALIDATE_EMAIL (int)
    Идентификатор фильтра "validate_email".
    FILTER_VALIDATE_IP (int)
    Идентификатор фильтра "validate_ip".
    FILTER_VALIDATE_MAC (int)
    Идентификатор фильтра "validate_mac_address".
    FILTER_DEFAULT (int)
    Идентификатор фильтра по умолчанию ("unsafe_raw"). Эквивалентно FILTER_UNSAFE_RAW.
    FILTER_UNSAFE_RAW (int)
    Идентификатор фильтра "unsafe_raw".
    FILTER_SANITIZE_STRING (int)
    Идентификатор фильтра "string". (Объявлен устаревшим, начиная с PHP 8.1.0, используйте вместо него функцию htmlspecialchars()).
    FILTER_SANITIZE_STRIPPED (int)
    Идентификатор фильтра "stripped". (Объявлен устаревшим, начиная с PHP 8.1.0, используйте вместо него функцию htmlspecialchars()).
    FILTER_SANITIZE_ENCODED (int)
    Идентификатор фильтра "encoded".
    FILTER_SANITIZE_SPECIAL_CHARS (int)
    Идентификатор фильтра "special_chars".
    FILTER_SANITIZE_EMAIL (int)
    Идентификатор фильтра "email".
    FILTER_SANITIZE_URL (int)
    Идентификатор фильтра "url".
    FILTER_SANITIZE_NUMBER_INT (int)
    Идентификатор фильтра "number_int".
    FILTER_SANITIZE_NUMBER_FLOAT (int)
    Идентификатор фильтра "number_float".
    FILTER_SANITIZE_MAGIC_QUOTES (int)
    Идентификатор фильтра "magic_quotes". (УСТАРЕЛ с PHP 7.3.0 и УДАЛЁН с PHP 8.0.0, используйте вместо него FILTER_SANITIZE_ADD_SLASHES.)
    FILTER_SANITIZE_ADD_SLASHES (int)
    Идентификатор фильтра "add_slashes". (Доступно с PHP 7.3.0)
    FILTER_CALLBACK (int)
    Идентификатор фильтра "callback".
    FILTER_FLAG_ALLOW_OCTAL (int)
    Разрешить восьмеричную запись (0[0-7]+) в фильтре "int".
    FILTER_FLAG_ALLOW_HEX (int)
    Разрешить шестнадцатеричную запись (0x[0-9a-fA-F]+) в фильтре "int".
    FILTER_FLAG_STRIP_LOW (int)
    Удалять символы с ASCII-кодом, меньшим 32.
    FILTER_FLAG_STRIP_HIGH (int)
    Удалять символы с ASCII-кодом, большим 127.
    FILTER_FLAG_STRIP_BACKTICK (int)
    Удаляет символы обратного ударения (грависа).
    FILTER_FLAG_ENCODE_LOW (int)
    Кодировать символы с ASCII-кодом, меньшим 32.
    FILTER_FLAG_ENCODE_HIGH (int)
    Кодировать символы с ASCII-кодом, большим 127.
    FILTER_FLAG_ENCODE_AMP (int)
    Кодировать &.
    FILTER_FLAG_NO_ENCODE_QUOTES (int)
    Не кодировать ' и ".
    FILTER_FLAG_EMPTY_STRING_NULL (int)
    (На данный момент не используется.)
    FILTER_FLAG_ALLOW_FRACTION (int)
    Разрешить дробную часть в фильтре "number_float".
    FILTER_FLAG_ALLOW_THOUSAND (int)
    Разрешить разделитель (,) в фильтре "number_float".
    FILTER_FLAG_ALLOW_SCIENTIFIC (int)
    Разрешить научную запись (e, E) в фильтре "number_float".
    FILTER_FLAG_PATH_REQUIRED (int)
    Требовать наличие пути в фильтре "validate_url".
    FILTER_FLAG_QUERY_REQUIRED (int)
    Требовать наличие строки запроса в фильтре "validate_url".
    FILTER_FLAG_SCHEME_REQUIRED (int)
    Требовать наличие схемы в фильтре "validate_url". (Объявлен устаревшим, начиная с PHP 7.3.0 и удалён в PHP 8.0.0, так как это и так подразумевается в фильтре).
    FILTER_FLAG_HOST_REQUIRED (int)
    Требовать наличие хоста в фильтре "validate_url". (Объявлен устаревшим, начиная с PHP 7.3.0 и удалён в PHP 8.0.0, так как это и так подразумевается в фильтре).
    FILTER_FLAG_HOSTNAME (int)
    Требовать, чтобы имена хостов начинались с буквенно-цифровых символов и содержали только буквенно-цифровые символы или дефисы. (Доступно с PHP 7.0.0)
    FILTER_FLAG_IPV4 (int)
    Разрешить только IPv4-адреса в фильтре "validate_ip".
    FILTER_FLAG_IPV6 (int)
    Разрешить только IPv6-адреса в фильтре "validate_ip".
    FILTER_FLAG_NO_RES_RANGE (int)
    Запретить зарезервированные адреса в фильтре "validate_ip".
    FILTER_FLAG_NO_PRIV_RANGE (int)
    Запретить частные адреса в фильтре "validate_ip".
    FILTER_FLAG_EMAIL_UNICODE (int)
    Принимать Unicode в локальной части фильтра "validate_email". (Доступно с PHP 7.1.0)
