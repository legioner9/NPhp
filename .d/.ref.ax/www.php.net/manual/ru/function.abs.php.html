    abs

    (PHP 4, PHP 5, PHP 7, PHP 8)

    abs — Возвращает абсолютную величину (модуль числа)

    Описание

    abs(int|float $num): int|float

    Возвращает абсолютную величину числа num.

    Список параметров

    num

    Числовое значение.

    Возвращаемые значения

    Возвращает абсолютную величину числа num. Если в параметр num передан аргумент в виде числа с плавающей точкой (float), функция также возвращает число с плавающей точкой (float), иначе — целое число (int) (поскольку у float часто больший диапазон значений, чем у int).

    Список изменений

    Версия Описание
    8.0.0 Параметр num больше не принимает внутренние объекты, поддерживающие числовое преобразование.

    Примеры

    Пример #1 Пример использования функции abs()

    <?php

    var_dump    (abs(-4.2));
    var_dump(abs(5));
    var_dump(abs(-5));
    ?>

    Результат выполнения приведённого примера: 

    float(4.2)
int(5)
int(5)
