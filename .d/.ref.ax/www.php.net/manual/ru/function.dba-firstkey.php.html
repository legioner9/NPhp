    dba_firstkey

    (PHP 4, PHP 5, PHP 7, PHP 8)

    dba_firstkey — Извлекает первый ключ

    Описание

    dba_firstkey(resource $dba): string|false

    dba_firstkey() возвращает первый ключ из базы данных и сбрасывает внутренний указатель. Это позволяет производить прямой поиск по всей базе.

    Список параметров

    dba

    Обработчик базы данных, возвращённый dba_open() или dba_popen().

    Возвращаемые значения

    Возвращает ключ в случае успешного выполнения или false в случае возникновения ошибки.

    Смотрите также

    • dba_nextkey() - Извлекает следующий ключ
    • dba_key_split() - Разделяет ключ, заданный в виде строки и создаёт массив из полученных частей
    • Второй пример в примерах DBA
