    dbase_delete_record

    (PHP 5 < 5.3.0, dbase 5, dbase 7)

    dbase_delete_record — Удаление записей из базы данных

    Описание

    dbase_delete_record(resource $database, int $number): bool

    Помечает указанные записи для удаления из базы данных.

    Замечание:

    Для окончательного удаления записей из базы данных, необходимо вызвать dbase_pack().

    Список параметров

    database

    Ресурс базы данных, возвращаемый функцией dbase_open() или dbase_create().

    number

    Целое число, в промежутке от 1 до количества записей в базе дынных (количество записей возвращает функция dbase_numrecords()).

    Возвращаемые значения

    Возвращает true в случае успешного выполнения или false в случае возникновения ошибки.

    Список изменений

    Версия Описание
    dbase 7.0.0 Параметр database теперь имеет тип resource а не int.

    Смотрите также

    • dbase_add_record() - Добавляет запись в базу данных
    • dbase_replace_record() - Заменяет запись в базе данных
