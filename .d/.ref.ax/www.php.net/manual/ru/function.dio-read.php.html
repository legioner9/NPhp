    dio_read

    (PHP 4 >= 4.2.0, PHP 5 < 5.1.0)

    dio_read — Прочитай байты из файлового дескриптора

    Описание

    dio_read(resource $fd, int $len = 1024): string

    Функция dio_read() читает и возвращает len байт из дескриптора fd.

    Список параметров

    fd

    Файловый дескриптор, полученный из dio_open().

    len

    Количество байт для чтения. Если не задано, то dio_read() прочитает блок в 1 килобайт.

    Возвращаемые значения

    Байты будут читаться из fd.

    Смотрите также

    • dio_write() - Записывает байты в файл, опционально обрезая до указанной длины
