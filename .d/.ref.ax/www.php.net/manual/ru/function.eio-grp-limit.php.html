    eio_grp_limit

    (PECL eio >= 0.0.1dev)

    eio_grp_limit — Устанавливает предельное количество запросов в группе

    Описание

    eio_grp_limit(resource $grp, int $limit): void

    Устанавливает предельное количество запросов в группе.

    Список параметров

    grp

    Указатель на группу ресурсов

    limit

    Максимальное количество ресурсов в группе.

    Возвращаемые значения

    Функция не возвращает значения после выполнения.
