    enchant_dict_is_added

    (PHP 8)

    enchant_dict_is_added — Определяет, существует ли слово в этой орфографической сессии

    Описание

    enchant_dict_is_added(EnchantDictionary $dictionary, string $word): bool

    Определяет, существует ли уже слово в текущей сессии.

    Список параметров

    dictionary

    Словарь Enchant, который возвращает функция enchant_broker_request_dict() или enchant_broker_request_pwl_dict().

    word

    Слово для поиска

    Возвращаемые значения

    Возвращает true, если слово уже существует, в противном случае возвращает false

    Список изменений

    Версия Описание
    8.0.0 dictionary ожидает экземпляр EnchantDictionary; Ранее ожидался ресурс (resource).
