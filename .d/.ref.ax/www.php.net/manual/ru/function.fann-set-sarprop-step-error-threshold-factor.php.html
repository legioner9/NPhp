    fann_set_sarprop_step_error_threshold_factor

    (PECL fann >= 1.0.0)

    fann_set_sarprop_step_error_threshold_factor — Устанавливает пороговый коэффициент ошибки шага sarprop

    Описание

    fann_set_sarprop_step_error_threshold_factor(resource $ann, float $sarprop_step_error_threshold_factor): bool

    Устанавливает пороговый коэффициент ошибки шага sarprop.

    Список параметров

    ann

    Ресурс (resource) нейронной сети.

    sarprop_step_error_threshold_factor

    Пороговый коэффициент ошибки шага sarprop.

    Возвращаемые значения

    Возвращает true в случае успешного выполнения, или false в противном случае.

    Примечания

    Замечание:

    Функция доступна только в том случае, если модуль fann был собран для libfann >= 2.2.

    Смотрите также
