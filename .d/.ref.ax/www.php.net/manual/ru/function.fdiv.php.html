    fdiv

    (PHP 8)

    fdiv — Делит одно число на другое по правилам стандарта IEEE 754

    Описание

    fdiv(float $num1, float $num2): float

    Возвращает в виде числа с плавающей точкой результат деления числа num1 на число num2. Если число num2 равно нулю, возвращает положительную бесконечность INF, отрицательную бесконечность -INF или нечисло NAN.

    Обратите внимание, что сравнение нечисла NAN с другими значениями, включая само нечисло, не будет истинным, ни при нестрогом сравнении ==, ни при строгом сравнении ===.

    Список параметров

    num1

    Делимое.

    num2

    Делитель.

    Возвращаемые значения

    Возвращает число с плавающей точкой — результат деления числа num1 на число num2.

    Примеры

    Пример #1 Пример использования функции fdiv()

    <?php

    var_dump    (fdiv(5.7, 1.3)); // float(4.384615384615385)
    var_dump(fdiv(4, 2)); // float(2)
    var_dump(fdiv(1.0, 0.0)); // float(INF)
    var_dump(fdiv(-1.0, 0.0)); // float(-INF)
    var_dump(fdiv(0.0, 0.0)); // float(NAN)

    ?>
