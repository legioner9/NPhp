    ftp_get_option

    (PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

    ftp_get_option — Получает текущие параметры FTP-соединения

    Описание

    ftp_get_option(FTP\Connection $ftp, int $option): int|bool

    Эта функция возвращает значение запрошенной опции option для указанного FTP-соединения.

    Список параметров

    ftp

    An FTP\Connection instance.

    option

    На текущий момент, поддерживаются следующие опции:

    Поддерживаемые текущие опции FTP
    FTP_TIMEOUT_SEC Возвращает текущее время ожидания, используемое в сетевых операциях.
    FTP_AUTOSEEK Возвращает true, если эта опция включена, иначе false.

    Возвращаемые значения

    Возвращает значение в случае успешного выполнения, или false, если указанная опция option не поддерживается. В последнем случае также вызывается предупреждение.

    Список изменений

    Версия Описание
    8.1.0 Параметр ftp теперь ожидает экземпляр FTP\Connection; ранее ожидался ресурс (resource).

    Примеры

    Пример #1 Пример использования ftp_get_option()

    <?php
    // Получаем время ожидания соединения
    $timeout = ftp_get_option($ftp, FTP_TIMEOUT_SEC);
    ?>

    Please note that you can use this function to get the value of FTP_USEPASSIVEADDRESS option also :

    <?php

    $conn     = ftp_connect("host");
    $login = ftp_login($conn, "user", "password");

    var_dump(ftp_get_option($conn, FTP_USEPASVADDRESS)); // true (which is the default value)

    ftp_set_option($conn, FTP_USEPASVADDRESS, false); // change the value

    var_dump(ftp_get_option($conn, FTP_USEPASVADDRESS)); // false

    ?>
