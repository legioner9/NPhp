    ftp_raw

    (PHP 5, PHP 7, PHP 8)

    ftp_raw — Отправляет произвольную команду FTP-серверу

    Описание

    ftp_raw(FTP\Connection $ftp, string $command): ?array

    Отправляет произвольную команду command FTP-серверу.

    Список параметров

    ftp

    An FTP\Connection instance.

    command

    Команда.

    Возвращаемые значения

    Возвращает ответ сервера в виде массива строк или null в случае возникновения ошибки. Функция ftp_raw() не интерпретирует ответ сервера и не определяет, успешно ли выполнена команда.

    Список изменений

    Версия Описание
    8.1.0 Параметр ftp теперь ожидает экземпляр FTP\Connection; ранее ожидался ресурс (resource).

    Примеры

    Пример #1 Использование ftp_raw() для входа на FTP-сервер

    <?php
    $ftp     = ftp_connect("ftp.example.com");

    /* То же самое, что:
    ftp_login($ftp, "joeblow", "secret"); */
    ftp_raw($ftp, "USER joeblow");
    ftp_raw($ftp, "PASS secret");
    ?>
