    get_declared_traits

    (PHP 5 >= 5.4.0, PHP 7, PHP 8)

    get_declared_traits — Возвращает массив со всеми объявленными трейтами

    Описание

    get_declared_traits(): array

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    Возвращает массив с именами трейтов в качестве значений массива.

    Смотрите также

    • class_uses() - Возвращает список трейтов, используемых заданным классом
    • trait_exists() - Проверяет, существует ли трейт

    Example of use:

    <?php

    namespace Example;

    // Declare Trait
    trait FooTrait
    {
    }

    // Declare Abstract class
    abstract class FooAbstract
    {
    }

    // Declare class
    class Bar extends FooAbstract
    {
    use     FooTrait;
    }

    // Get all traits declareds
    $array = get_declared_traits();

    var_dump($array);
    /**
    * Result:

    * array(1) {
    * [0] =>
    * string(23) "Example\FooTrait"
    * }
    */
