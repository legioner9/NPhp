    getrandmax

    (PHP 4, PHP 5, PHP 7, PHP 8)

    getrandmax — Возвращает максимально возможное случайное число

    Описание

    getrandmax(): int

    Возвращает максимальное значение, которое может быть получено функцией rand().

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    Максимально возможное случайное значение, возвращаемое функцией rand()

    Смотрите также

    • rand() - Генерирует случайное число
    • srand() - Задаёт начальное число генератора псевдослучайных чисел
    • mt_getrandmax() - Показывает максимально возможное значение случайного числа
