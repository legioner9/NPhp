    gmp_perfect_power

    (PHP 7 >= 7.3.0, PHP 8)

    gmp_perfect_power — Проверить, является ли число "совершенной степенью"

    Описание

    gmp_perfect_power(GMP|int|string $num): bool

    Проверяет, является ли num совершенной степенью. Целое число является совершенной степенью, если его можно получить возведением меньшего целого числа в целую степень.

    Список параметров

    num

    Объект GMP, целое число (int) или числовая строка (string).

    Возвращаемые значения

    Возвращает true, если num является совершенной степенью. В противном случае возвращает false.
