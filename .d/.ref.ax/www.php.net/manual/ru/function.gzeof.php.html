    gzeof

    (PHP 4, PHP 5, PHP 7, PHP 8)

    gzeof — Проверяет, находится ли текущая позиция в конце (EOF) gz-файла

    Описание

    gzeof(resource $stream): bool

    Проверяет, соответствует ли позиция указателя в GZ-файле позиции EOF (end-of-file, конец файла).

    Список параметров

    stream

    Указатель на gz-файл. Он должен быть корректный и должен указывать на файл, успешно открытый gzopen().

    Возвращаемые значения

    Возвращает true, если позиция указателя gz-файла соответствует EOF или возникает ошибка; иначе возвращает false.

    Примеры

    Пример #1 Пример использования gzeof()

    <?php
    $gz     = gzopen('somefile.gz', 'r');
    while (!    gzeof($gz)) {
    echo     gzgetc($gz);
    }
    gzclose($gz);
    ?>
