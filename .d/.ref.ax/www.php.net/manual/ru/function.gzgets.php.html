    gzgets

    (PHP 4, PHP 5, PHP 7, PHP 8)

    gzgets — Получить строку из указателя файла

    Описание

    gzgets(resource $stream, ?int $length = null): string|false

    Получает строку (несжатую) из gz-файла, её длина ограничивается length - 1 байтами. Чтение заканчивается при достижении максимальной длины, конца строки или конца файла (EOF), в зависимости от того, что наступит раньше.

    Список параметров

    stream

    Указатель на gz-файл. Он должен быть корректный и должен указывать на файл, успешно открытый gzopen().

    length

    Максимальный размер возвращаемых данных.

    Возвращаемые значения

    Распакованная строка или false в случае возникновения ошибки.

    Список изменений

    Версия Описание
    8.0.0 length теперь допускает значение null; ранее значение по умолчанию было 1024.

    Примеры

    Пример #1 Пример использования gzgets()

    <?php
    $handle     = gzopen('somefile.gz', 'r');
    while (!    gzeof($handle)) {
    $buffer = gzgets($handle, 4096);
    echo     $buffer;
    }
    gzclose($handle);
    ?>

    Смотрите также

    • gzopen() - Открыть gz-файл
    • gzgetc() - Получить символ из указателя на gz-файл
    • gzwrite() - Бинарная запись в gz-файл
