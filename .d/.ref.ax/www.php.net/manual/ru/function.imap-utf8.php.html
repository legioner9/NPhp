    imap_utf8

    (PHP 4, PHP 5, PHP 7, PHP 8)

    imap_utf8 — Преобразовывает MIME-кодированный текст в UTF-8

    Описание

    imap_utf8(string $mime_encoded_text): string

    Преобразовывает заданную в параметре mime_encoded_text строку в UTF-8, если заявленная кодировка известна модулю libc-client. В противном случае данный текст декодируется, но не преобразовывается в UTF-8.

    Список параметров

    mime_encoded_text

    MIME-кодированная строка. Метод кодирования MIME и спецификация UTF-8 описаны в » RFC2047 и » RFC2044 соответственно.

    Возвращаемые значения

    Возвращает декодированную строку, если возможно, преобразованную в UTF-8.

    Примеры

    Пример #1 Базовое использование imap_utf8()

    <?php
    echo imap_utf8("Johannes =?ISO-8859-1?Q?Schl=FCter?=");
    ?>

    Вывод приведённого примера будет похож на: 

    Johannes Schlüter

    To get a complete decode to utf8 (which is the protocol with "national" characters), use these two:
    $subject = "=?ISO-8859-1?Q?B=F8lla?=, med =?ISO-8859-1?Q?=F8l?= i baggen";
    $subject = imap_utf8($subject);
    $subject = utf8_decode($subject);
    will return the subject in 8-bit ascii.
    That is, both functions.
