    ldap_first_entry

    (PHP 4, PHP 5, PHP 7, PHP 8)

    ldap_first_entry — Возвратить первый идентификатор результата

    Описание

    ldap_first_entry(LDAP\Connection $ldap, LDAP\Result $result): LDAP\ResultEntry|false

    Возвращает идентификатор для первой записи в результате. Этот идентификатор затем используется в функции ldap_next_entry(), для последовательного получения записей из результата.

    Записи в LDAP результате считываются последовательно используя функции ldap_first_entry() и ldap_next_entry().

    Список параметров

    ldap

    Экземпляр LDAP\Connection, который возвращает функция ldap_connect().

    result

    Экземпляр LDAP\Result, который возвращает функция ldap_list() или ldap_search().

    Возвращаемые значения

    Возвращает экземпляр LDAP\ResultEntry или false в случае возникновения ошибки.

    Список изменений

    Версия Описание
    8.1.0 Параметр ldap теперь ожидает экземпляр LDAP\Connection; ранее ожидался ресурс (resource) ldap link .
    8.1.0 Параметр result теперь ожидает экземпляр LDAP\Result; ранее ожидался ресурс (resource) ldap result.
    8.1.0 Возвращает экземпляр LDAP\ResultEntry; ранее возвращался ресурс (resource).
