    mb_ereg_search_regs

    (PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

    mb_ereg_search_regs — Возвращает совпавшую с регулярным выражением часть строки

    Описание

    mb_ereg_search_regs(?string $pattern = null, ?string $options = null): array|false

    Возвращает совпавшую с регулярным выражением часть строки.

    Список параметров

    pattern

    Шаблон поиска.

    options

    Вариант поиска. Объяснение дано в описании функции mb_regex_set_options().

    Возвращаемые значения

    Возвращает массив (array), в первом элементе которого будет подстрока совпавшей части, во втором сгруппированная в скобках первая часть, в третьем сгруппированная в скобках вторая часть и так далее, если будет найдено соответствие регулярному выражению строки в многобайтовой кодировке. В случае возникновения ошибки возвращает false.

    Список изменений

    Версия Описание
    8.0.0 Параметры pattern и options теперь могут принимать значение null.

    Примечания

    Замечание:

    Для этой функции будет использована внутренняя кодировка или кодировка, установленная функцией mb_regex_encoding().

    Смотрите также

    • mb_regex_encoding() - Устанавливает/получает кодировку символов для многобайтового регулярного выражения
    • mb_ereg_search_init() - Настраивает строку и регулярное выражение для поиска соответствий строки в многобайтовой кодировке регулярному выражению
