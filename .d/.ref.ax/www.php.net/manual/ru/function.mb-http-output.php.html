    mb_http_output

    (PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

    mb_http_output — Устанавливает/получает кодировку символов вывода HTTP

    Описание

    mb_http_output(?string $encoding = null): string|bool

    Устанавливает/получает кодировку символов HTTP-вывода. Выходные данные после работы этой функции будут преобразованы из внутренней кодировки к значению параметра encoding.

    Список параметров

    encoding

    Если задан параметр encoding, функция mb_http_output() устанавливает кодировку выходных символов HTTP в значение параметра encoding.

    Если аргумент encoding опущен, функция mb_http_output() возвращает текущую кодировку символов вывода HTTP.

    Возвращаемые значения

    Если аргумент encoding опущен, функция mb_http_output() возвращает текущую кодировку символов вывода HTTP. В противном случае Возвращает true в случае успешного выполнения или false в случае возникновения ошибки.

    Список изменений

    Версия Описание
    8.0.0 Теперь параметр encoding может принимать значение null.
