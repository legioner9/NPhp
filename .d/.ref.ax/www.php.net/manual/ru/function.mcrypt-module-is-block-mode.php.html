    mcrypt_module_is_block_mode

    (PHP 4 >= 4.0.2, PHP 5, PHP 7 < 7.2.0, PECL mcrypt >= 1.0.0)

    mcrypt_module_is_block_mode — Проверить, возвращает ли указанный режим данные блоками или нет

    Внимание

    Эта функция объявлена УСТАРЕВШЕЙ начиная с PHP 7.1.0 и была УДАЛЕНА в версии PHP 7.2.0. Использовать эту функцию крайне не рекомендуется.

    Описание

    mcrypt_module_is_block_mode(string $mode, string $lib_dir = ?): bool

    Функция возвращает true, если данные возвращаются блоками и false, если побайтно. (то есть true для cbc и ecb, и false для cfb и потока).

    Список параметров

    mode

    Одна из констант MCRYPT_MODE_modename, либо одна из следующих строк: "ecb", "cbc", "cfb", "ofb", "nofb" и "stream".

    lib_dir

    Опциональный параметр lib_dir, в котором можно указать директорию, содержащую модуль алгоритма.

    Возвращаемые значения

    Функция возвращает true, если данные возвращаются блоками и false если побайтно. (то есть true для cbc и ecb, и false для cfb и потока).
