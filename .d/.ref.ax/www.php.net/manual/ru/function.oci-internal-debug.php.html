    oci_internal_debug

    (PHP 5, PHP 7, PECL OCI8 >= 1.1.0, PECL OCI8 <= 2.2.0)

    oci_internal_debug — Включает или отключает внутренний вывод отладки

    Описание

    oci_internal_debug(bool $onoff): void

    Включает или отключает внутренний вывод отладки.

    Список параметров

    onoff

    Установите значение false для отключения вывода отладки или true, чтобы включить.

    Возвращаемые значения

    Функция не возвращает значения после выполнения.

    Примечания

    Замечание:

    Функция была удалена в PHP 8 и PECL OCI8 3.0.
