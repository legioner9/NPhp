    odbc_connection_string_should_quote

    (PHP 8 >= 8.2.0)

    odbc_connection_string_should_quote — Определяет, следует ли заключать значение строки соединения ODBC в кавычки

    Описание

    odbc_connection_string_should_quote(string $str): bool

    Определяет, нужно ли заключать строку в кавычки для строкового значения соединения ODBC, т.е. содержит ли она специальные символы.

    Обратите внимание, что при этом не проверяется, заключена ли строка в кавычки; уже заключённая в кавычки строка будет содержать символы, которые заставят эту функцию вернуть значение true. Для проверки следует использовать функцию odbc_connection_string_is_quoted().

    Список параметров

    str

    Строка, которую необходимо проверить.

    Возвращаемые значения

    Возвращает true, если строка должна быть заключена в кавычки, в противном случае возвращает false.

    Смотрите также

    • odbc_connection_string_quote() - Заключает в кавычки значение строки подключения
    • odbc_connection_string_is_quoted() - Определяет, заключено ли значение строки соединения ODBC в кавычки
