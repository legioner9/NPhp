    pcntl_wstopsig

    (PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

    pcntl_wstopsig — Получить сигнал, из-за которого был остановлен дочерний процесс

    Описание

    pcntl_wstopsig(int $status): int|false

    Возвращает сигнал, из-за которого был остановлен дочерний процесс. Функция имеет смысл только если pcntl_wifstopped() вернула true.

    Список параметров

    status

    Параметр status — это параметр статуса, передаваемый для успешного вызова функции pcntl_waitpid().

    Возвращаемые значения

    Возвращает номер сигнала. Если эта функция не поддерживается ОС, возвращается false.

    Смотрите также

    • pcntl_waitpid() - Ожидает или возвращает статус порождённого дочернего процесса
    • pcntl_wifstopped() - Проверить, остановлен ли дочерний процесс
