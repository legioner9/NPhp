    pg_connection_status

    (PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

    pg_connection_status — Определяет состояние подключения

    Описание

    pg_connection_status(PgSql\Connection $connection): int

    pg_connection_status() возвращает состояние переданного в качестве аргумента соединения connection.

    Список параметров

    connection

    Экземпляр PgSql\Connection.

    Возвращаемые значения

    PGSQL_CONNECTION_OK либо PGSQL_CONNECTION_BAD.

    Список изменений

    Версия Описание
    8.1.0 Параметр connection теперь ожидает экземпляр PgSql\Connection; ранее ожидался ресурс (resource).

    Примеры

    Пример #1 Пример использования pg_connection_status()

    <?php
    $dbconn     = pg_connect("dbname=publisher") or die("Не удалось соединиться");
    $stat = pg_connection_status($dbconn);
    if (    $stat === PGSQL_CONNECTION_OK) {
    echo     'Статус соединения: доступно';
    } else {
    echo     'Статус соединения: разорвано';
    }
    ?>

    Смотрите также

    • pg_connection_busy() - Проверяет, занято ли соединение в данный момент.

    It returns null, but you have to be careful with your comparison operators.

    <?php
    unset($null);
    if (    pg_connection_status($null)===PGSQL_CONNECTION_OK)
    echo     'this is not called';
    if (    pg_connection_status($null)==PGSQL_CONNECTION_OK)
    echo     'this is called because NULL==0 is true';
    ?>
