    pow

    (PHP 4, PHP 5, PHP 7, PHP 8)

    pow — Возводит в степень

    Описание

    pow(mixed $num, mixed $exponent): int|float|object

    Возвращает число num в степени exponent.

    Замечание:

    Допустимо указывать оператор ** вместо вызова функции.

    Список параметров

    num

    Основание.

    exponent

    Показатель степени.

    Возвращаемые значения

    Возвращает основание num в степени exponent. Если оба аргумента — неотрицательные целые и результат можно представить как целое число, то функция возвращает целое число (int). В противном случае функция возвращает число с плавающей точкой (float).

    Примеры

    Пример #1 Примеры использования функции pow()

    <?php

    var_dump    (pow(2, 8)); // int(256)
    echo pow(-1, 20); // 1
    echo pow(0, 0); // 1
    echo pow(10, -1); // 0.1

    echo pow(-1, 5.5); // NAN

    ?>

    Примечания

    Замечание:

    Функция преобразует входные значения, даже нескалярные, в числовые, что иногда даёт странный результат.

    Смотрите также

    • exp() - Вычисляет степень числа e
    • sqrt() - Извлекает квадратный корень
    • bcpow() - Возведение в степень чисел с произвольной точностью
    • gmp_pow() - Возводит число в степень

    Many notations use "^" as a power operator, but in PHP (and other C-based languages) that is actually the XOR operator. You need to use this 'pow' function, there is no power operator.

    i.e. 3^2 means "3 XOR 2" not "3 squared".

    It is particular confusing as when doing Pythagoras theorem in a 'closet points' algorithm using "^" you get results that look vaguely correct but with an error.
