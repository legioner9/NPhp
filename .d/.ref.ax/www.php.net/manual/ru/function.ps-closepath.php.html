    ps_closepath

    (PECL ps >= 1.1.0)

    ps_closepath — Замыкает путь

    Описание

    ps_closepath(resource $psdoc): bool

    Соединяет последнюю точку с первой точкой пути. Полученный контур можно использовать для обводки, заливки, обрезки и т.д.

    Список параметров

    psdoc

    Идентификатор ресурса файла postscript, возвращённый функцией ps_new().

    Возвращаемые значения

    Возвращает true в случае успешного выполнения или false в случае возникновения ошибки.

    Смотрите также

    • ps_clip() - Отрисовка клипов по текущему пути
    • ps_closepath_stroke() - Замыкает и обводит контур
