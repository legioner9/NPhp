    rnp_save_keys

    (PECL rnp >= 0.1.1)

    rnp_save_keys — Сохраняет ключи в строке PHP

    Описание

    rnp_save_keys(
        RnpFFI $ffi,
        string $format,
        string &$output,
        int $flags
    ): bool

    Обратите внимание, что для G10 значением output должен быть каталог (который уже должен существовать).

    Список параметров

    ffi

    Объект FFI, который возвращает функция rnp_ffi_create.

    format

    Формат ключа данных (GPG, KBX, G10).

    output

    Ключевые пакеты будут сохранены в строке, на которую ссылается output.

    flags

    Смотрите описание флагов RNP_LOAD_SAVE_*.

    Возвращаемые значения

    Возвращает true в случае успешного выполнения или false в случае возникновения ошибки.
