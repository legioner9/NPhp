    ssh2_poll

    (PECL ssh2 >= 0.9.0)

    ssh2_poll — Опрашивает каналы/слушателей/потоки на предмет событий

    Описание

    ssh2_poll(array &$desc, int $timeout = 30): int

    Опрашивает каналы/слушателей/потоки на предмет событий и возвращает количество дескрипторов, которые вернули ненулевые повторения.

    Внимание

    Функция пока не документирована; для знакомства доступен только список аргументов.

    Список параметров

    desc

    Индексированный массив подмассивов с ключами 'resource' и 'events'. Значение ресурса - это (канал) поток или ресурс SSH2 Listener. Значением события являются флаги SSH2_POLL*, объединённые побитовым ИЛИ. Каждый подмассив будет заполнен элементом 'revents' при возврате, значения которого представляют собой флаги SSH2_POLL*, объединённые побитовым ИЛИ совместно с произошедшими событиями.

    timeout

    Время ожидания в секундах.

    Возвращаемые значения

    Возвращает количество дескрипторов, которые вернули ненулевые повторения.
