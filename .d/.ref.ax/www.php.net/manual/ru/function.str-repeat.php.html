    str_repeat

    (PHP 4, PHP 5, PHP 7, PHP 8)

    str_repeat — Возвращает повторяющуюся строку

    Описание

    str_repeat(string $string, int $times): string

    Возвращает строку string, повторенную times раз.

    Список параметров

    string

    Повторяемая строка.

    times

    Количество раз, которые нужно повторить строку string.

    times должен быть больше или равен нулю. Если times равен нулю, возвращается пустая строка.

    Возвращаемые значения

    Возвращает повторяющуюся строку.

    Примеры

    Пример #1 Пример использования str_repeat()

    <?php
    echo str_repeat("-=", 10);
    ?>

    Результат выполнения приведённого примера: 

    -=-=-=-=-=-=-=-=-=-=

    Here is a simple one liner to repeat a string multiple times with a separator:

    <?php
    implode    ($separator, array_fill(0, $multiplier, $input));
    ?>

    Example script:
    <?php

    // How I like to repeat a string using standard PHP functions
    $input = 'bar';
    $multiplier = 5;
    $separator = ',';
    print     implode($separator, array_fill(0, $multiplier, $input));
    print     "\n";

    // Say, this comes in handy with count() on an array that we want to use in an
    // SQL query such as 'WHERE foo IN (...)'
    $args = array('1', '2', '3');
    print     implode(',', array_fill(0, count($args), '?'));
    print     "\n";
    ?>

    Example Output:
    bar,bar,bar,bar,bar
    ?,?,?
