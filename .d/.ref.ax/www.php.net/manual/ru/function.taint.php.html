    taint

    (PECL taint >=0.1.0)

    taint — Испортить строку

    Описание

    taint(string &$string, string ...$strings): bool

    Делает строку испорченной строку. Используется в целях отладки.

    Список параметров

    string

    strings

    Возвращаемые значения

    Возвращает true, если преобразование строки выполнено. Всегда возвращает true, если модуль taint не включён.
