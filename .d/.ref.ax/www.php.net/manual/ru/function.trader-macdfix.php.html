    trader_macdfix

    (PECL trader >= 0.2.0)

    trader_macdfix — Исправление схождения/расхождения скользящей средней 12/26

    Описание

    trader_macdfix(array $real, int $signalPeriod = ?): array

    Список параметров

    real

    Массив, содержащий реальные значения.

    signalPeriod

    Сглаживание для сигнальной линии (номер периода). Допустимы значения от 1 до 100000.

    Возвращаемые значения

    Возвращает массив с вычисленными данными или false в случае возникновения ошибки.
