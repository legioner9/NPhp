    wddx_add_vars

    (PHP 4, PHP 5, PHP 7)

    wddx_add_vars — Добавить переменные в пакет WDDX с указанным идентификатором

    Внимание

    Эта функция была УДАЛЕНА в PHP 7.4.0.

    Описание

    wddx_add_vars(resource $packet_id, mixed $var_name, mixed ...$var_names): bool

    Сериализует переданные переменные и добавляет результат к данному пакету.

    Список параметров

    Функция принимает переменное количество параметров.

    packet_id

    Пакет WDDX, возвращаемый wddx_packet_start().

    var_name

    Может быть либо строкой, обозначающей переменную, либо массивом, содержащий строки c именами переменных или другие массивы и т.д.

    var_names

    Возвращаемые значения

    Возвращает true в случае успешного выполнения или false в случае возникновения ошибки.

    Here is a simple way of serializing dynamic values from an array:

    <?PHP
    //a unix timestamp
    $date = "1094095513";

    //some data to be included
    $books = array('programming' => array('php','perl','java'),
    'markup' => array('UML','XML','HTML')
    );

    //stick data to an array to iterate over
    $data_to_serialize = array($date,$books);

    //create the packet
    $packet = wddx_packet_start("SOME DATA ARRAY");

    //loop through the data
    foreach($data_to_serialize as $key => $data)
    {
    //create a var whith the name of the content of $key
    $$key = $data;
    wddx_add_vars($packet,$key);
    }

    echo     wddx_packet_end($packet);

    ?>

    Have fun !
