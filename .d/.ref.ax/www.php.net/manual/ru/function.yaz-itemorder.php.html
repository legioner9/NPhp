    yaz_itemorder

    (PHP 4 >= 4.0.5, PECL yaz >= 0.9.0)

    yaz_itemorder — Подготавливает запрос к Z39.50 Item Order с пакетом ILL-Request

    Описание

    yaz_itemorder(resource $id, array $args): void

    Функция подготавливает запрос Extended Services с использованием профиля для использования Item Order Extended Service для Transport ILL (Profile/1). Смотрите » это и » спецификацию.

    Список параметров

    id

    Ресурс подключения, возвращаемый yaz_connect().

    args

    Должен быть ассоциативным массивом с информацией об отправляемом запросе Item Order. Ключ хеша - это имя соответствующего пути тега ASN.1. Например, у ISBN под Item-ID будет ключ item-id,ISBN.
