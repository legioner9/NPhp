    GearmanWorker::setId

    (No version information available, might only be in Git)

    GearmanWorker::setId — Назначает обработчику идентификатор, чтобы в дальнейшем иметь возможность опросить все доступные обработчики

    Описание

    public GearmanWorker::setId(string $id): bool

    Назначает обработчику идентификатор.

    Список параметров

    id

    Строковый идентификатор.

    Возвращаемые значения

    Возвращает true в случае успешного выполнения или false в случае возникновения ошибки.

    Примеры

    Пример #1 Пример использования GearmanWorker::setId()

    Задание идентификатора для простого обработчика.

    <?php
    $worker    = new GearmanWorker();
    $worker->setId('test');
    ?>

    Вывод приведённого примера будет похож на: 

    Run the following command:
gearadmin --workers

Output:
30 ::3a3a:3361:3361:3a33%976303667 - : test
