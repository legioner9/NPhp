    Предопределённые константы

    Модуль определяет перечисленные ниже константы и открывает к ним доступ только тогда, когда модуль либо собрали в PHP, либо динамически загрузили во время исполнения кода.

    GD_VERSION (string)
    Версия GD, с которой был скомпилирован PHP.
    GD_MAJOR_VERSION (int)
    Полнофункциональная версия GD, с которой был скомпилирован PHP.
    GD_MINOR_VERSION (int)
    Сокращённая версия GD, с которой был скомпилирован PHP.
    GD_RELEASE_VERSION (int)
    Рабочая версия GD, с которой был скомпилирован PHP.
    GD_EXTRA_VERSION (string)
    Дополнительная (beta/rc..) версия GD, с которой был скомпилирован PHP.
    GD_BUNDLED (int)
    Если используется распространяемая с PHP версия GD, константа принимает значение 1, в противном случае 0.
    IMG_AVIF (int)
    Возвращается как значение функции imagetypes(). (Доступно, начиная с PHP 8.1.0)
    IMG_BMP (int)
