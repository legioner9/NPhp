    Imagick::clone

    (PECL imagick 2, PECL imagick 3)

    Imagick::clone — Создаёт точную копию объекта Imagick

    Описание

    public Imagick::clone(): Imagick

    Создаёт точную копию объекта Imagick.

    Внимание

    Функция устарела (DEPRECATED) с imagick версии 3.1.0, взамен неё используйте ключевое слово clone.

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    Будет возвращена копия объекта Imagick.

    Список изменений

    Версия Описание
    PECL imagick 3.1.0 Метод устарел в пользу ключевого слова clone.

    Примеры

    Пример #1 Клонирование объекта Imagick в разных версиях

    <?php
    // Клонирование объекта Imagick в imagick 2.x и 3.0:
    $newImage = $image->clone();

    // Клонирование объекта Imagick с версии 3.1.0:
    $newImage = clone $image;
    ?>
