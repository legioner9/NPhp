    Предопределённые константы

    Модуль определяет перечисленные ниже константы и открывает к ним доступ только тогда, когда модуль либо собрали в PHP, либо динамически загрузили во время исполнения кода.

    Константы COLOR_*

    Константы цветовых типов. Эти константы используются в основном с ImagickPixel.

    imagick::COLOR_BLACK (int)
    Чёрный
    imagick::COLOR_BLUE (int)
    Синий
    imagick::COLOR_CYAN (int)
    Сине-зелёный
    imagick::COLOR_GREEN (int)
    Зелёный
    imagick::COLOR_RED (int)
    Красный
    imagick::COLOR_YELLOW (int)
    Жёлтый
    imagick::COLOR_MAGENTA (int)
    Пурпурный
    imagick::COLOR_OPACITY (int)
    Прозрачность цвета
    imagick::COLOR_ALPHA (int)
    Альфа-канал для цвета
    imagick::COLOR_FUZZ (int)
    'Fuzz factor' для цвета. Смотрите документацию по ImageMagick.

    Константы оператора наложения
    imagick::COMPOSITE_DEFAULT (int)
    Оператор наложения по умолчанию
    imagick::COMPOSITE_UNDEFINED (int)
    Неопределённый оператор наложения
    imagick::COMPOSITE_NO (int)
    Не установленный оператор наложения
    imagick::COMPOSITE_ADD (int)
    Результатом будет изображение + изображение
    imagick::COMPOSITE_ATOP (int)
    Результатом будет изображение такой же формы с наложением изображения затемняя его в местах перекрытия
    imagick::COMPOSITE_BLEND (int)
    Смесь изображения
    imagick::COMPOSITE_BUMPMAP (int)
    Так же, как COMPOSITE_MULTIPLY, за исключением того, что источник сначала преобразуется в оттенки серого.
    imagick::COMPOSITE_CLEAR (int)
    Делает целевое изображение прозрачным
    imagick::COMPOSITE_COLORBURN (int)
    Затемняет конечное изображение с учётом исходного изображения
    imagick::COMPOSITE_COLORDODGE (int)
    Повышенная яркость конечного изображения с учётом исходного изображения
    imagick::COMPOSITE_COLORIZE (int)
    Раскраска целевого изображения с помощью составного изображения
    imagick::COMPOSITE_COPYBLACK (int)
    Копирует чёрный цвет из исходного изображения в целевое
    imagick::COMPOSITE_COPYBLUE (int)
    Копирует синий цвет из исходного изображения в целевое
    imagick::COMPOSITE_COPY (int)
    Копирует исходное изображение в целевое
    imagick::COMPOSITE_COPYCYAN (int)
    Копирует сине-зелёный цвет из исходного изображения в целевое
    imagick::COMPOSITE_COPYGREEN (int)
    Копирует зелёный цвет из исходного изображения в целевое
    imagick::COMPOSITE_COPYMAGENTA (int)
    Копирует пурпурный цвет из исходного изображения в целевое
    imagick::COMPOSITE_COPYOPACITY (int)
    Копирует прозрачность из исходного изображения в целевое
    imagick::COMPOSITE_COPYRED (int)
    Копирует красный цвет из исходного изображения в целевое
    imagick::COMPOSITE_COPYYELLOW (int)
    Копирует жёлтый цвет из исходного изображения в целевое
    imagick::COMPOSITE_DARKEN (int)
    Затемняет целевое изображение
    imagick::COMPOSITE_DSTATOP (int)
    Область назначения, которая находится внутри исходного изображения накладывается поверх исходного и заменяет назначение
    imagick::COMPOSITE_DST (int)
    Целевое изображение остаётся нетронутым
    imagick::COMPOSITE_DSTIN (int)
    Область внутри исходного изображения заменяется заданным изображением
    imagick::COMPOSITE_DSTOUT (int)
    Область снаружи исходного изображения заменяется заданным изображением
    imagick::COMPOSITE_DSTOVER (int)
    Исходное изображение заменяется заданным
    imagick::COMPOSITE_DIFFERENCE (int)
    Вычитание более тёмного цвета из двух составляющих
    imagick::COMPOSITE_DISPLACE (int)
    Смещение пикселей заданного изображения по аналогии с исходным изображением
    imagick::COMPOSITE_DISSOLVE (int)
    Растворение исходного изображения в заданном
    imagick::COMPOSITE_EXCLUSION (int)
    Создание эффекта, похожего на imagick::COMPOSITE_DIFFERENCE, но с более низким контрастом
    imagick::COMPOSITE_HARDLIGHT (int)
    Умножает или осветляет цвет, в зависимости от исходного значения цвета
    imagick::COMPOSITE_HUE (int)
    Изменение оттенка заданного изображения по аналогии с исходным изображением
    imagick::COMPOSITE_IN (int)
    Наложение исходного изображения на заданное
    imagick::COMPOSITE_LIGHTEN (int)
    Освещает заданное изображение по аналогии с исходным изображением
    imagick::COMPOSITE_LUMINIZE (int)
    Изменение яркости заданного изображения по аналогии с исходным изображением
    imagick::COMPOSITE_MINUS (int)
    Вычитает исходное изображение из заданного
    imagick::COMPOSITE_MODULATE (int)
    Модулирует целевые яркость, насыщенности и тон по аналогии с исходным изображением
    imagick::COMPOSITE_MULTIPLY (int)
    Перемножает цель и источник
    imagick::COMPOSITE_OUT (int)
    Наложение внешних частей источника на цель
    imagick::COMPOSITE_OVER (int)
    Наложение источника на цель
    imagick::COMPOSITE_OVERLAY (int)
    Преобразование "Overlay"
    imagick::COMPOSITE_PLUS (int)
    Добавляет источник к цели
    imagick::COMPOSITE_REPLACE (int)
    Заменяет цель источником
    imagick::COMPOSITE_SATURATE (int)
    Меняет насыщенность цели в соответствии с источником
    imagick::COMPOSITE_SCREEN (int)
    Источник и цель сначала дополняются, потом перемножаются, а потом источник заменяет цель.
    imagick::COMPOSITE_SOFTLIGHT (int)
    Затемняет или осветляет цвета, в зависимости от источника
    imagick::COMPOSITE_SRCATOP (int)
    Часть источника, лежащая внутри цели, накладывается на цель
    imagick::COMPOSITE_SRC (int)
    Источник копируется в цель
    imagick::COMPOSITE_SRCIN (int)
    Часть источника, лежащая внутри цели, заменяет цель
    imagick::COMPOSITE_SRCOUT (int)
    Часть источника, лежащая вне цели, заменяет цель
    imagick::COMPOSITE_SRCOVER (int)
    Источник заменяет цель
    imagick::COMPOSITE_SUBTRACT (int)
    Вычитает цвета источника из цели
    imagick::COMPOSITE_THRESHOLD (int)
    Источник накладывается на цель как определено порогом источника
    imagick::COMPOSITE_XOR (int)
    Часть источника, лежащая вне цели комбинируется с частью цели, лежащей вне источника

    The `COLOR_OPACITY` doesn't exist if you are using ImageMagick lib 7 or newer version!

    Here is the c source code:

    #if MagickLibVersion < 0x700
    IMAGICK_REGISTER_CONST_LONG("COLOR_OPACITY", PHP_IMAGICK_COLOR_OPACITY);
    #endif

    There isn't any information about it anywhere (or I didn't find it)! So, if you want to use it, use the 6 version of lib! You can see the current installed version of lib with this command:

    $ identify -version

    I believe the description for imagick::COMPOSITE_MINUS is incorrect. It states:

    "Subtracts the source from the target"

    But according to ImageMagick's documentation:

    "The original method name 'Minus' is a synonym for 'Minus_Dst' meaning 'minus the destination'"

    And defines "Minus_Dst" as:
    "Src - Dest"

    So the description of "imagick::COMPOSITE_MINUS" should be "Subtracts the target from the source"
