    Предопределённые константы

    Модуль определяет перечисленные ниже константы и открывает к ним доступ только тогда, когда модуль либо собрали в PHP, либо динамически загрузили во время исполнения кода.

    NIL (int)
    Объявлена устаревшей, начиная с PHP 8.1.0.
    OP_DEBUG (int)
    OP_READONLY (int)
    Открыть почтовый ящик только для чтения
    OP_ANONYMOUS (int)
    Не используйте обновления .newsrc для новостей (только для NNTP)
    OP_SHORTCACHE (int)
    OP_SILENT (int)
    OP_PROTOTYPE (int)
    OP_HALFOPEN (int)
    Для IMAP и NNTP имён, открыть соединение, но не открывать почтовый ящик.
    OP_EXPUNGE (int)
    OP_SECURE (int)
    CL_EXPUNGE (int)
    Не выдавая никаких предупреждений стирать сообщения прежде чем закрыть при вызове imap_close()
    FT_UID (int)
    Параметром является UID
    FT_PEEK (int)
    Не устанавливать флаг \Seen, если до этого он ещё не установлен
    FT_NOT (int)
    FT_INTERNAL (int)
    Возвращаемая строка во внутреннем формате. Не будет канонизирована к виду CRLF.
    FT_PREFETCHTEXT (int)
    ST_UID (int)
    Последовательность аргументов, которая содержит идентификаторы UID вместо последовательности номеров
    ST_SILENT (int)
    ST_SET (int)
    CP_UID (int)
    порядковые номера, содержащие идентификаторы UID
    CP_MOVE (int)
    Удалить сообщения из текущего почтового ящика после копирования с помощью imap_mail_copy()
    SE_UID (int)
    Возвращать идентификаторы UID вместо порядковых номеров
    SE_FREE (int)
    SE_NOPREFETCH (int)
    Не выполнять предвыборку найденных сообщений
    SO_FREE (int)
    SO_NOSERVER (int)
    SA_MESSAGES (int)
    SA_RECENT (int)
    SA_UNSEEN (int)
    SA_UIDNEXT (int)
    SA_UIDVALIDITY (int)
    SA_ALL (int)
    LATT_NOINFERIORS (int)
    Этот почтовый ящик не имеет "потомков". (Почтовые ящики за этим отсутствуют)
    LATT_NOSELECT (int)
    Это только контейнер, а не почтовый ящик. Вы не можете открыть его.
    LATT_MARKED (int)
    Этот почтовый ящик помечен. Используется только UW-IMAPD.
    LATT_UNMARKED (int)
    Этот почтовый ящик не помечен. Используется только с UW-IMAPD.
    LATT_REFERRAL (int)
    Этот контейнер имеет направления (referral) на удалённый почтовый ящик.
    LATT_HASCHILDREN (int)
    Этот почтовый ящик имеет выбираемые подчинённые (inferiors).
    LATT_HASNOCHILDREN (int)
    Этот почтовый ящик не имеет выбираемые подчинённые (inferiors).
    SORTDATE (int)
    Критерий сортировки для imap_sort(): дата написания
    SORTARRIVAL (int)
    Критерий сортировки для imap_sort(): дата доставки
    SORTFROM (int)
    Критерий сортировки для imap_sort(): почтовый ящик первый в поле "от кого" (поле from)
    SORTSUBJECT (int)
    Критерий сортировки для imap_sort(): тема письма
    SORTTO (int)
    Критерий сортировки для imap_sort(): почтовый ящик первый в поле "кому" (поле to)
    SORTCC (int)
    Критерий сортировки для imap_sort(): почтовый ящик первый в поле "копия" (поле cc)
    SORTSIZE (int)
    Критерий сортировки для imap_sort(): размер сообщения в байтах
    TYPETEXT (int)
    Первичный тип тела сообщения: неформатированный текст
    TYPEMULTIPART (int)
    Первичный тип тела сообщения: несколько частей
    TYPEMESSAGE (int)
    Первичный тип тела сообщения: инкапсулированное сообщение
    TYPEAPPLICATION (int)
    Первичный тип тела сообщения: данные приложения
    TYPEAUDIO (int)
    Первичный тип тела сообщения: аудио
    TYPEIMAGE (int)
    Первичный тип тела сообщения: статическое изображение
    TYPEVIDEO (int)
    Первичный тип тела сообщения: видео
    TYPEMODEL (int)
    Первичный тип тела сообщения: модель
    TYPEOTHER (int)
    Первичный тип тела сообщения: неизвестно
    ENC7BIT (int)
    Кодировка тела сообщения: 7-битные данные в семантике SMTP
    ENC8BIT (int)
    Кодировка тела сообщения: 8-битные данные в семантике SMTP
    ENCBINARY (int)
    Кодировка тела сообщения: 7-битные двоичные данные
    ENCBASE64 (int)
    Кодировка тела сообщения: данные закодированные в base-64
    ENCQUOTEDPRINTABLE (int)
    Кодировка тела сообщения: человеко-читаемые 7-как-8 битные данные
    ENCOTHER (int)
    Кодировка тела сообщения: неизвестно
    IMAP_OPENTIMEOUT (int)
    IMAP_READTIMEOUT (int)
    IMAP_WRITETIMEOUT (int)
    IMAP_CLOSETIMEOUT (int)
    IMAP_GC_ELT (int)
    Сборщик мусора, очищает элементы кеша сообщений.
    IMAP_GC_ENV (int)
    Сборщик мусора, очищает обёртку и тело сообщения.
    IMAP_GC_TEXTS (int)
    Сборщик мусора, очищает текст.
