    IntlChar::ord

    (PHP 7, PHP 8)

    IntlChar::ord — Получить код символ Unicode

    Описание

    public static IntlChar::ord(int|string $character): ?int

    Возвращает код символа Unicode.

    Эта функция противоположна по действию функции IntlChar::chr().

    Список параметров

    character

    Целочисленное (int) задание кода символа (например 0x2603 для U+2603 СНЕГОВИКА), или символ закодированный строку UTF-8 (например "\u{2603}")

    Возвращаемые значения

    Возвращает код символа Unicode в виде целого числа.

    Примеры

    Пример #1 Тестирование различных способов задания

    <?php
    var_dump    (IntlChar::ord("A"));
    var_dump(IntlChar::ord(" "));
    var_dump(IntlChar::ord("\u{2603}"));
    ?>

    Результат выполнения приведённого примера: 

    int(65)
int(32)
int(9731)

    Смотрите также

    • IntlChar::chr() - Получить символ Unicode по его коду
    • mb_ord() - Получает кодовую точку символа Unicode
    • ord() - Конвертирует первый байт строки в число от 0 до 255
