    Введение

    Модуль mysqli позволяет вам получить доступ к функциональности, которую предоставляет MySQL версии 4.1 и выше. Больше информации о сервере баз данных MySQL вы можете получить по адресу » http://www.mysql.com/.

    Информация об использовании MySQL из PHP находится по адресу Краткий обзор

    Документация MySQL находится по адресу » http://dev.mysql.com/doc/.

    Некоторые части данной документации включены из руководства по MySQL с разрешения корпорации Oracle.

    Все примеры, приведённые в документации, используют доступные свободно базы данных » world и » sakila.
