    Установка

    Модуль JSON по умолчанию входит в поставку PHP.

    Начиная с PHP 8.0.0, модуль JSON является основным модулем PHP, поэтому оно всегда включён.

    Информация по установке этого модуля PECL может быть найдена в главе руководства Установка PECL модулей. Дополнительная информация, такая как новые версии, скачивание, исходные файлы, информация о разработчике и CHANGELOG, может быть найдена здесь: » https://pecl.php.net/package/json

    ; extension=json.so

    must be uncommented (delete the semicolon) and Apache restarted for these functions to be available.
