    Установка для PHP >= 7.4

    Модуль libxml включён по умолчанию, но может быть отключён при указании директивы --without-libxml.

    PHP использует утилиту pkg-config для выбора правильного файла библиотеки, заголовочных файлов и флагов компиляции для работы с модулем libxml2. Чтобы убедиться, что выбрана желаемая версия модуля libxml2, можно перед запуском сценария настройки через переменную окружения PKG_CONFIG_PATH указать утилите pkg-config путь для поиска нужной версии модуля: 

    PKG_CONFIG_PATH="/path/to/libxml2/prefix/lib/pkgconfig:/lib/pkgconfig"
