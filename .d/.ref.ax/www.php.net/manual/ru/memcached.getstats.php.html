    Memcached::getStats

    (PECL memcached >= 0.1.0)

    Memcached::getStats — Получает статистику о серверах в пуле

    Описание

    public Memcached::getStats(?string $type = null): array|false

    Memcached::getStats() возвращает массив, содержащий статистику всех доступных memcache серверов. Ознакомьтесь также со спецификацией » memcache протокола для получения полной информации об этой статистике.

    Список параметров

    type

    Тип статистики, которую необходимо получить.

    Возвращаемые значения

    Список массивов со статистикой по каждому серверу, где каждый элемент - отдельный сервер или false в случае возникновения ошибки.

    Примеры

    Пример #1 Пример использования Memcached::getStats()

    <?php
    $m     = new Memcached();
    $m->addServer('localhost', 11211);

    print_r($m->getStats());
    ?>
