    MongoDB\Driver\BulkWrite::insert

    (mongodb >=1.0.0)

    MongoDB\Driver\BulkWrite::insert — Добавить операцию вставки в порцию

    Описание

    public MongoDB\Driver\BulkWrite::insert(array|object $document): mixed

    Добавляем операцию вставки в MongoDB\Driver\BulkWrite.

    Список параметров

    document (array|object)

    Документ для вставки.

    Возвращаемые значения

    Возвращает _id добавленного документа. Если в document не был задан _id, будет возвращён MongoDB\BSON\ObjectId, сгенерированный для вставки.

    Ошибки

    • При ошибке парсинга аргумента бросает исключение MongoDB\Driver\Exception\InvalidArgumentException.

    Список изменений

    Версия Описание
    PECL mongodb 1.3.0 Всегда возвращается _id для добавленного документа. Ранее метод возвращал значение только в случае если MongoDB\BSON\ObjectId был создан.

    Примеры

    Пример #1 Пример использования MongoDB\Driver\BulkWrite::insert()

    <?php

    $bulk     = new MongoDB\Driver\BulkWrite;

    $document1 = ['title' => 'one'];
    $document2 = ['_id' => 'custom ID', 'title' => 'two'];
    $document3 = ['_id' => new MongoDB\BSON\ObjectId, 'title' => 'three'];

    $_id1 = $bulk->insert($document1);
    $_id2 = $bulk->insert($document2);
    $_id3 = $bulk->insert($document3);

    var_dump($_id1, $_id2, $_id3);

    $manager = new MongoDB\Driver\Manager('mongodb://localhost:27017');
    $result = $manager->executeBulkWrite('db.collection', $bulk);

    ?>

    Вывод приведённого примера будет похож на: 

    object(MongoDB\BSON\ObjectId)#3 (1) {
  ["oid"]=>
  string(24) "54d51146bd21b91405401d92"
}
