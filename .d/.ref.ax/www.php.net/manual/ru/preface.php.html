    Предисловие

    PHP, что означает "PHP: Hypertext Preprocessor" - «PHP: Препроцессор Гипертекста», — распространённый интерпретируемый язык общего назначения с открытым исходным кодом, который создавался специально для ведения веб-разработок, и код на нём встраивается непосредственно в HTML-код. Синтаксис языка берёт начало из языков C, Java и Perl и лёгок для изучения. Основная цель языка — помочь веб-разработчикам создавать динамически генерируемые веб-страницы быстро, но с PHP можно сделать существенно больше.

    Это руководство в основном состоит из справочника по функциям, но также содержит справочник языка, объяснения наиболее важных особенностей PHP и другие дополнительные сведения.

    Руководство доступно в нескольких форматах по адресу » https://www.php.net/download-docs.php. Подробнее о том, как ведётся работа над руководством, рассказано в приложении «Об этом руководстве». Тем, кому интересна история языка, обращаются к разделу «История PHP».
