    ReflectionClass::__construct

    (PHP 5, PHP 7, PHP 8)

    ReflectionClass::__construct — Создаёт объект класса ReflectionClass

    Описание

    public ReflectionClass::__construct(object|string $objectOrClass)

    Создаёт новый объект класса ReflectionClass.

    Список параметров

    objectOrClass

    В качестве аргумента может принимать строку (string), содержащую имя исследуемого класса, либо объект (object).

    Ошибки

    Вызывает исключение ReflectionException, если заданный класс не существует.

    Примеры

    Пример #1 Простой пример использования ReflectionClass

    <?php
    $reflection     = new ReflectionClass('Exception');
    echo     $reflection;
    ?>

    To reflect on a namespaced class in PHP 5.3, you must always specify the fully qualified name of the class - even if you've aliased the containing namespace using a "use" statement.

    So instead of:

    <?php
    use App\Core as Core;
    $oReflectionClass = new ReflectionClass('Core\Singleton');
    ?>

    You would type:

    <?php
    use App\Core as Core;
    $oReflectionClass = new ReflectionClass('App\Core\Singleton');
    ?>
