    ReflectionFunction::export

    (PHP 5, PHP 7)

    ReflectionFunction::export — Экспортирует функции

    Внимание

    Эта функция УСТАРЕЛА начиная с PHP 7.4.0 и была УДАЛЕНА в PHP 8.0.0. Использовать эту функцию крайне не рекомендуется.

    Описание

    public static ReflectionFunction::export(string $name, string $return = ?): string

    Экспортирует отражённую (reflected) функцию.

    Список параметров

    name

    Экспортируемый объект Reflection.

    return

    Установка в true вернёт экспортируемое значение, в отличие от поведения, где этот параметр опущен. Установка в false (по умолчанию) сделает противоположное.

    Возвращаемые значения

    Если параметр return установлен в true, тогда экспортируемый объект будет возвращён как string, иначе будет возвращён null.

    Смотрите также

    • ReflectionFunction::invoke() - Вызывает функцию
    • ReflectionFunction::__toString() - Возвращает строковое представление объекта ReflectionFunction
