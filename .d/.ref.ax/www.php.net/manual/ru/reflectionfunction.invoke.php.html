    ReflectionFunction::invoke

    (PHP 5, PHP 7, PHP 8)

    ReflectionFunction::invoke — Вызывает функцию

    Описание

    public ReflectionFunction::invoke(mixed ...$args): mixed

    Вызывает отражённую (reflected) функцию.

    Список параметров

    args

    Передаваемые функции аргументы. Может принимать переменное количество аргументов по аналогии с call_user_func().

    Возвращаемые значения

    Возвращает результат выполнения вызванной функции.

    Примеры

    Пример #1 Пример использования ReflectionFunction::invoke()

    <?php
    function title($title, $name)
    {
    return     sprintf("%s. %s\r\n", $title, $name);
    }

    $function = new ReflectionFunction('title');

    echo     $function->invoke('Dr', 'Phil');
    ?>

    Результат выполнения приведённого примера: 

    Dr. Phil

    Примечания

    Замечание:

    ReflectionFunction::invoke() не может использоваться, если ожидаются ссылочные параметры. Вместо него следует использовать ReflectionFunction::invokeArgs() (передача ссылок в списке аргументов).

    Смотрите также

    • ReflectionFunction::export() - Экспортирует функции
    • __invoke()
    • call_user_func() - Вызывает callback-функцию, заданную в первом параметре

    I know Reflections classes have a lot of power, but sometimes all we need is to store a annonymus function or even create a simple callback somewhere.

    so here it is, the Callback class:
    <?php
    class Callback{
    private     $name = false;

    public function     Callback($obj,$call=false){
    $name = array($obj);
    if(    $call) $name[] = $call;

    $this->name = $name;
    }
    public function     invoke($params=array()){
    return     call_user_func_array($this->name,$params);
    }
    }
    ?>

    Usage:
    <?php
    function sayName(){
    return     "goku";
    }

    $myVar = new Callback("sayName");

    echo     "Hi, I am ".$myVar->invoke()."!";

    //also works with methods: new Callback($obj,"method");
    ?>
