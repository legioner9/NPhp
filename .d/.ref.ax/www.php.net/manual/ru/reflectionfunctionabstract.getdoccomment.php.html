    ReflectionFunctionAbstract::getDocComment

    (PHP 5 >= 5.1.0, PHP 7, PHP 8)

    ReflectionFunctionAbstract::getDocComment — Получает doc-комментарий

    Описание

    public ReflectionFunctionAbstract::getDocComment(): string|false

    Получает doc-комментарий из описания функции.

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    Комментарий, если он есть, false в противном случае.
