    ReflectionNamedType::isBuiltin

    (PHP 7, PHP 8)

    ReflectionNamedType::isBuiltin — Проверяет, является ли тип встроенным

    Описание

    public ReflectionNamedType::isBuiltin(): bool

    Проверяет, является ли тип встроенным типом PHP. Встроенный тип - любой тип, не являющийся классом, интерфейсом или трейтом.

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    true, если тип встроенный, false в противном случае.

    Примеры

    Пример #1 Пример использования ReflectionNamedType::isBuiltin()

    <?php
    class SomeClass {}

    function     someFunction(string $param, SomeClass $param2, stdClass $param3) {}

    $reflectionFunc = new ReflectionFunction('someFunction');
    $reflectionParams = $reflectionFunc->getParameters();

    var_dump($reflectionParams[0]->getType()->isBuiltin());
    var_dump($reflectionParams[1]->getType()->isBuiltin());
    var_dump($reflectionParams[2]->getType()->isBuiltin());

    Результат выполнения приведённого примера: 

    bool(true)
bool(false)
bool(false)

    Заметьте, что метод ReflectionNamedType::isBuiltin() не различает внутренние и пользовательские классы. Чтобы их различить, нужно использовать метод ReflectionClass::isInternal() для возвращаемого имени класса.

    Смотрите также

    • ReflectionType::allowsNull() - Проверяет, допустим ли NULL
    • ReflectionType::__toString() - Преобразование в строку
    • ReflectionClass::isInternal() - Проверяет, является ли класс встроенным в модуль или в ядро
    • ReflectionParameter::getType() - Получить тип параметра

    Throwable
    IteratorAggregate
    ArrayAccess
    WeakReference
