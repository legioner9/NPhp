    ReflectionParameter::export

    (PHP 5, PHP 7)

    ReflectionParameter::export — Экспорт

    Внимание

    Эта функция УСТАРЕЛА начиная с PHP 7.4.0 и была УДАЛЕНА в PHP 8.0.0. Использовать эту функцию крайне не рекомендуется.

    Описание

    public static ReflectionParameter::export(string $function, string $parameter, bool $return = ?): string

    Экспорт.

    Внимание

    Функция пока не документирована; для знакомства доступен только список аргументов.

    Список параметров

    function

    Имя функции.

    parameter

    Имя аргумента.

    return

    Установка в true вернёт экспортируемое значение, в отличие от поведения, где этот параметр опущен. Установка в false (по умолчанию) сделает противоположное.

    Возвращаемые значения

    Экспорт отражения (reflection).
