    SolrClient::deleteById

    (PECL solr >= 0.9.2)

    SolrClient::deleteById — Удаляет по идентификатору

    Описание

    public SolrClient::deleteById(string $id): SolrUpdateResponse

    Удаляет документ с указанным идентификатором. Где ID - значение поля uniqueKey, объявленного в схеме.

    Список параметров

    id

    Значение поля uniqueKey, объявленное в схеме

    Возвращаемые значения

    Возвращает объект SolrUpdateResponse или выбрасывает исключение в случае возникновения ошибки.

    Ошибки

    Выбрасывает SolrClientException, если клиент отказал или возникла проблема с подключением.

    Выбрасывает SolrServerException, если сервер Solr не смог обработать запрос.

    Смотрите также

    • SolrClient::deleteByIds() - Удаляет по идентификаторам
    • SolrClient::deleteByQuery() - Удаляет все документы, соответствующие заданному запросу
    • SolrClient::deleteByQueries() - Удаляет все документы, соответствующие любому из запросов
