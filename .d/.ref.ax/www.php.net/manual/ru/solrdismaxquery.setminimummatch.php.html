    SolrDisMaxQuery::setMinimumMatch

    (No version information available, might only be in Git)

    SolrDisMaxQuery::setMinimumMatch — Устанавливает минимальное соответствие "Should" (mm)

    Описание

    public SolrDisMaxQuery::setMinimumMatch(string $value): SolrDisMaxQuery

    Устанавливает параметр минимального соответствия "Should" (mm). Если оператор запроса по умолчанию - AND (И), тогда mm = 100%, если оператор запроса по умолчанию (q.op) - OR (ИЛИ), то mm = 0%.

    Список параметров

    value

    Минимальное значение соответствия/выражение.

    Возвращаемые значения

    SolrDisMaxQuery

    Примеры

    Пример #1 Пример использования SolrDisMaxQuery::setMinimumMatch()

    <?php

    $dismaxQuery     = new SolrDisMaxQuery("lucene");
    // 75% предложений запроса должны соответствовать
    $dismaxQuery->setMinimumMatch("75%");
    echo     $dismaxQuery . PHP_EOL;
    ?>

    Результат выполнения приведённого примера: 

    q=lucene&defType=edismax&mm=75%
