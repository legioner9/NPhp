    SplFileInfo::getSize

    (PHP 5 >= 5.1.2, PHP 7, PHP 8)

    SplFileInfo::getSize — Получает размер файла

    Описание

    public SplFileInfo::getSize(): int|false

    Возвращает размер файла в байтах.

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    Размер файла в байтах в случае успешного выполнения или false в случае возникновения ошибки.

    Ошибки

    Будет выброшено исключение RuntimeException, если файл не существует или возникла ошибка.

    Примеры

    Пример #1 Пример использования SplFileInfo::getSize()

    <?php
    $info     = new SplFileInfo('example.jpg');
    echo     $fileinfo->getFilename() . " " . $fileinfo->getSize();
    ?>

    Вывод приведённого примера будет похож на: 

    example.jpg 15385
