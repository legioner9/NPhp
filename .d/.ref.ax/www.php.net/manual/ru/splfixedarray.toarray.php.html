    SplFixedArray::toArray

    (PHP 5 >= 5.3.0, PHP 7, PHP 8)

    SplFixedArray::toArray — Возвращает обычный PHP-массив со значениями фиксированного массива

    Описание

    public SplFixedArray::toArray(): array

    Возвращает обычный PHP-массив со значениями фиксированного массива.

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    Возвращает PHP-массив (array), составленный из значений фиксированного массива.

    Примеры

    Пример #1 Пример использования SplFixedArray::toArray()

    <?php
    $fa     = new SplFixedArray(3);
    $fa[0] = 0;
    $fa[2] = 2;
    var_dump($fa->toArray());
    ?>

    Результат выполнения приведённого примера: 

    array(3) {
  [0]=>
  int(0)
  [1]=>
  NULL
  [2]=>
  int(2)
}
