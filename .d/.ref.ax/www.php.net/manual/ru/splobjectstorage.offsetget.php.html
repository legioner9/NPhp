    SplObjectStorage::offsetGet

    (PHP 5 >= 5.3.0, PHP 7, PHP 8)

    SplObjectStorage::offsetGet — Возвращает данные, ассоциированные с объектом object

    Описание

    public SplObjectStorage::offsetGet(object $object): mixed

    Возвращает данные, ассоциированные с объектом object.

    Список параметров

    object

    Искомый объект object.

    Возвращаемые значения

    Данные, ассоциированные с объектом object.

    Ошибки

    Выбрасывает исключение UnexpectedValueException, если объект object не удалось найти.

    Примеры

    Пример #1 Пример использования SplObjectStorage::offsetGet()

    <?php
    $s     = new SplObjectStorage;

    $o1 = new stdClass;
    $o2 = new stdClass;

    $s[$o1] = "hello";
    $s->attach($o2);


    var_dump($s->offsetGet($o1)); // $s[$o1]
    var_dump($s->offsetGet($o2)); // $s[$o2]
    ?>

    Вывод приведённого примера будет похож на: 

    string(5) "hello"
NULL

    Смотрите также

    • SplObjectStorage::offsetSet() - Ассоциирует данные с объектом в контейнере
    • SplObjectStorage::offsetExists() - Проверяет, существует ли объект в контейнере
    • SplObjectStorage::offsetUnset() - Удаляет объект из контейнера

    // SplObjectStorage
    $object = new stdClass;
    $test = new SplObjectStorage;

    $start = microtime(true);
    for (    $i = 0; $i < 1000000; $i++) {
    $test->offsetGet($object);

    // Array + spl_object_hash()
    $object = new stdClass;
    $test = [];

    $start = microtime(true);
    for (    $i = 0; $i < 1000000; $i++) {
    $temp = $test[spl_object_hash($object)];
    }
