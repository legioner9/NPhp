    Предопределённые константы

    Модуль определяет перечисленные ниже константы и открывает к ним доступ только тогда, когда модуль либо собрали в PHP, либо динамически загрузили во время исполнения кода.

    SQLITE3_ASSOC (int)
    Указывает на то, что Sqlite3Result::fetchArray() метод должен вернуть массив, индексированный по имени столбца в соответствующий набор результатов.
    SQLITE3_NUM (int)
    Указывает на то, что Sqlite3Result::fetchArray() метод должен вернуть массив, индексированный по номеру столбца в соответствующий набор результатов, начиная со столбца 0.
    SQLITE3_BOTH (int)
    Указывает на то, что Sqlite3Result::fetchArray() метод должен вернуть массив, индексированный по имени столбца и его номеру в соответствующий набор результатов, начиная со столбца 0.
    SQLITE3_INTEGER (int)
    Представляет класс хранения типа SQLite3 INTEGER.
    SQLITE3_FLOAT (int)
    Представляет класс хранения типа SQLite3 REAL (FLOAT).
    SQLITE3_TEXT (int)
    Представляет класс хранения типа SQLite3 TEXT.
    SQLITE3_BLOB (int)
    Представляет класс хранения типа SQLite3 BLOB.
    SQLITE3_NULL (int)
    Представляет класс хранения типа SQLite3 NULL.
    SQLITE3_OPEN_READONLY (int)
    Указывает на то, что база данных SQLite3 доступна только для чтения.
    SQLITE3_OPEN_READWRITE (int)
    Указывает на то, что база данных SQLite3 доступна как для чтения, так и для записи.
    SQLITE3_OPEN_CREATE (int)
    Указывает на то, что файл базы данных SQLite3 будет создан в том случае, если он ещё не существует.
    SQLITE3_DETERMINISTIC (int)
    Указывает, что функция созданная SQLite3::sqliteCreateFunction() детерминирована. Т.е. она всегда возвращает одинаковый результат для для одинаковых входных данных в одном выражении SQL. (Доступно с PHP 7.1.4.)
