    Thread::getThreadId

    (PECL pthreads >= 2.0.0)

    Thread::getThreadId — Идентификация

    Описание

    public Thread::getThreadId(): int

    Возвращает идентификатор указанного потока

    Список параметров

    У этой функции нет параметров.

    Возвращаемые значения

    Числовой идентификатор.

    Примеры

    Пример #1 Возвращает идентификатор указанного потока

    <?php
    class My extends Thread {
    public function     run() {
    printf("%s является потоком #%lu\n", __CLASS__, $this->getThreadId());
    }
    }
    $my = new My();
    $my->start();
    ?>

    Результат выполнения приведённого примера: 

    My является потоком #123456778899
