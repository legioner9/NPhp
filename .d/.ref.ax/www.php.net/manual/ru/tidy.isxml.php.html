    tidy::isXml

    tidy_is_xml

    (PHP 5, PHP 7, PHP 8, PECL tidy >= 0.5.2)

    tidy::isXml -- tidy_is_xml — Определяет, является ли документ общим XML-документом (не HTML/XHTML)

    Описание

    Объектно-ориентированный стиль

    public tidy::isXml(): bool

    Процедурный стиль

    tidy_is_xml(tidy $tidy): bool

    Определяет, является ли документ общим XML-документом (не HTML/XHTML).

    Список параметров

    tidy

    Объект Tidy.

    Возвращаемые значения

    Эта функция возвращает true, если указанный tidy-объект tidy является общим XML-документом (не HTML/XHTML), или false в противном случае.

    Внимание

    Эта функция ещё не реализована в самом Tidylib, поэтому всегда возвращается false.
