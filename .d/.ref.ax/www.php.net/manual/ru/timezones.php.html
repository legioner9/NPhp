    Здесь вы найдёте полный список часовых поясов, поддерживаемых PHP, которые могут быть использованы, например, в date_default_timezone_set().

    Предостережение

    Поведение часовых поясов, отсутствующих в списке, не определено.

    Замечание: Последняя версия базы данных часовых поясов может быть установлена с помощью PECL-пакета » timezonedb.

    Замечание: Этот список основан на версии базы данных часовых поясов 2024.1.
