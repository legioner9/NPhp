    Установка

    Этот модуль » PECL не поставляется вместе с PHP.

    Информация по установке этого модуля PECL может быть найдена в главе руководства Установка PECL модулей. Дополнительная информация, такая как новые версии, скачивание, исходные файлы, информация о разработчике и CHANGELOG, может быть найдена здесь: » https://pecl.php.net/package/yaf.

    Бинарные файлы Windows (файлы DLL) для этого модуля PECL доступны на сайте PECL.
